# cvrp-scalability-analysis
Este proyecto resuelve el Problema de Rutas de Vehículos con Capacidad (CVRP) mediante programación lineal entera mixta y la formulación MTZ. Utiliza PuLP para la optimización y genera visualizaciones automáticas con NetworkX, analizando cómo escala la complejidad computacional al incrementar el número de nodos en escenarios logísticos.
